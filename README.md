JevalCDN

Esté codigo es 100% codigo abierto con la intencion de poder hostear un cdn basado en apis para multiples propocitos.

Casos de uso:
Imaginate que tienes una pagina web con un sistema de creaciones de cuentas. quieres agregar que la gente quierra poner su foto de perfil, pero no tienes suficiente espacio en la matriz raiz o tampoco quieres que cualquier persona pueda vulnerar con archivos que puedan corromper o simplemente no quieres llenar el espacio del codigo con archivos de imagenes.
Esté sistema está hecho para correr en una maquina con su propia memoria y su propio puerto, puede usarse en proxmox o en otra pc.

RECOMENDADO Un servidor con debian , php y una base de datos sql con mariadb


Está permitido usar el codigo , sea comercial o personal siempré y cuando se cumpla las directices del GNU.


Recomendaciones:
Recomendamos encarecidamente usar nginx , apache funciona igual pero puede tener un bajon de rendimiento.
